# ec-price-research-tool
# EC価格リサーチ支援ツール（半自動・設計原型）

## 概要
EC事業者向けに、価格リサーチ〜価格改定判断を効率化するための  
**半自動型ツールの設計原型**です。

楽天市場・Yahoo!ショッピング・Amazon等で販売している商品について、
自社価格と競合価格の差額を整理し、  
**価格改定の判断材料を一覧で把握できる状態**を目的としています。

本ツールは「完全自動」ではなく、  
**安定運用・規約順守を前提とした現実的な自動化レベル**を想定しています。

---

## 想定用途
- 価格改定前のリサーチ業務の効率化
- 手作業による価格比較の削減
- EC運用における判断材料の整理
- 半自動ツールから段階的な自動化への移行

---

## 想定構成
- **入力**  
  - CSV または Googleスプレッドシート  
  - 商品名 / 商品URL / 自社価格 など

- **処理**  
  - 競合価格の取得（API / スクレイピング / 手動補助の組み合わせ）
  - 最安価格の抽出
  - 自社価格との差額計算
  - 条件に応じた判定フラグ付与

- **出力**  
  - 差額・判定結果を一覧化したCSV  
  - Googleスプレッドシートへの反映を想定

---

## 自動化レベルについて
本ツールは以下の考え方で設計しています。

- 初期段階では **半自動運用**
- 規約やアクセス制限を考慮した安全設計
- 業務負荷を減らすことを最優先
- 必要に応じて自動化範囲を段階的に拡張

「すべてを自動化する」のではなく、  
**実運用で無理のない形を優先**しています。

## 使用技術（想定）

---

## 想定処理フロー
1. 商品情報CSVを入力
2. 競合価格を取得（取得方法は要件に応じて切替）
3. 自社価格との差額を計算
4. 改定判断用フラグを付与
5. 結果をCSVとして出力
